About HeartFL

Heart Disease Prediction using Federated Learning

Project Overview

HeartFL is a cutting-edge healthcare application that leverages Federated Learning to predict heart disease risk while maintaining complete patient data privacy. Unlike traditional machine learning approaches that require centralized data collection, our system enables multiple hospitals to collaboratively train a shared model without ever sharing raw patient data.

Key Features

  • Privacy-preserving machine learning
  • Multi-hospital collaboration
  • Real-time heart disease risk assessment
  • Secure dataset management
  • Interactive FL dashboard
  • Prediction history tracking

What is Federated Learning?

Federated Learning (FL) is a machine learning technique that trains an algorithm across multiple decentralized edge devices or servers holding local data samples, without exchanging them.

FL Process in HeartFL:
  1. Dataset Upload: Each hospital uploads its patient dataset (stored locally)
  2. Local Training: Each hospital trains a model on its own data
  3. Model Update Sharing: Only model weights/parameters are shared with the central server
  4. Global Aggregation: Central server aggregates all model updates
  5. Distribution: Updated global model is sent back to all hospitals
Privacy Advantage: Patient data never leaves the hospital premises. Only encrypted model parameters are shared, ensuring HIPAA compliance and data privacy.
