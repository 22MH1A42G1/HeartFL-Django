What happens in a FL Round?
  1. Each hospital with a ready dataset trains a local logistic regression model
  2. Local model weights are sent to the server (no patient data is shared)
  3. Weights are aggregated using weighted averaging (FedAvg)
  4. A global model is created and stored
  5. Doctors can use this global model for predictions
